# 프로젝트 소개
DotheG; do the Green

친환경 활동도 게임처럼, 캐릭터 수집으로 만드는 나만의 Eco-Life!


# Backend

## ✨Main 기능
- 회원가입/로그인
- 목적에 맞는 중간지점 찾기
- 커뮤니티 게시판
- 즐겨찾기 장소/친구
